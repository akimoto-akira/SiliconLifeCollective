## 이 변경 로그에 대하여

### 프로젝트 기원

- 이 프로젝트는 2026년 3월 20일에 시작되었습니다.
- 이 프로젝트 이전에, 비합리적인 아키텍처 설계로 인해 실패한 검증 데모가 있었고, 여러 AI 플랫폼과 통합할 수 없었습니다.

### 사용한 AI IDE 도구

#### Kiro (Amazon AWS)
- 프로젝트는 처음에 Kiro로 유지 관리되었으며 Spec 모드로 시작했습니다.
- Kiro는 Amazon AWS에서 구축한 agentic AI 개발 환경입니다.
- Code OSS(VS Code)를 기반으로 하며, VS Code 설정과 Open VSX 호환 플러그인을 지원합니다.
- 구조화된 AI 코딩을 위한 사양 기반 개발 워크플로우를 제공합니다.

#### Comate AI IDE / 文心快码 (Baidu)
- 카피라이팅 및 문서 작업에 가끔 사용합니다.
- Comate AI IDE는 Baidu Wenxin이 2025년 6월 23일에 출시한 AI 네이티브 개발 환경 도구입니다.
- 업계 최초의 멀티모달, 멀티에이전트 협업 AI IDE입니다.
- 디자인에서 코드로의 변환과 전체 프로세스 AI 지원 코딩 기능을 포함합니다.
- Baidu Wenxin 4.0 X1 Turbo 모델로 구동됩니다.

#### Trae (ByteDance)
- 이 프로젝트의 대부분의 시간은 주로 Trae를 사용하여 유지 관리되었습니다.
- Trae는 ByteDance의 싱가포르 자회사 SPRING PTE에서 개발한 AI IDE입니다.
- 10x AI Engineer로서, 독립적으로 소프트웨어 솔루션을 구축할 수 있습니다.
- 지능형 생산성 도구, 유연한 개발 리듬 적응, 협업 프로젝트 전달 기능을 제공합니다.
- 구성 가능한 에이전트 시스템으로 엔터프라이즈급 성능을 제공합니다.

#### Qoder (Alibaba)
- 2026년 4월 18일부터 이 프로젝트는 Qoder를 사용하여 유지 관리되고 있습니다.
- Qoder는 소스 코드 분석 및 도메인 문서 생성에 뛰어나며, 복잡한 코드베이스 이해에 탁월한 성능을 발휘합니다.
- 제로 계산 비용 가격 모델을 채택하여 자동화된 문서 처리 및 일상적인 작업 처리에 매우 비용 효율적입니다.
- 실제 소프트웨어 개발을 위해 설계된 AI 기반 agentic 코딩 플랫폼입니다.
- 지능형 코드 생성, 대화형 프로그래밍, 고급 컨텍스트 분석 엔진, 멀티에이전트 협업 기능을 제공합니다.
- 최소한의 리소스 소비로 깊은 코드 이해를 제공하며, 장기 프로젝트 유지 관리 및 지식 축적에 이상적입니다.

### 요구사항 문서

- 이 프로젝트의 요구사항 문서는 공개되어 있지 않습니다.
- 요구사항은 12개 이상의 국제 AI 플랫폼 및 대규모 모델 시리즈를 통해 반복 검증되었으며, 2000줄 이상의 거의 인간이 이해할 수 없는 사용자 스토리 기반 요구사항 문서가 생성되었습니다.
