# Web UI 가이드

[English](../en/web-ui-guide.md) | [中文](../zh-CN/web-ui-guide.md) | [繁體中文](../zh-HK/web-ui-guide.md) | [Español](../es-ES/web-ui-guide.md) | [日本語](../ja-JP/web-ui-guide.md) | **한국어** | [Deutsch](../de-DE/web-ui-guide.md) | [Čeština](../cs-CZ/web-ui-guide.md)

## 개요

Web UI는 실리콘 생명체 관리, 시스템 상태 모니터링 및 AI 에이전트와의 상호작용을 위한 포괄적인 인터페이스를 제공합니다. 시스템은 순수 서버 사이드 렌더링 아키텍처를 채택하여 프론트엔드 프레임워크 의존성 없이 `H`, `CssBuilder`, `JsBuilder` 빌더를 통해 HTML, CSS 및 JavaScript를 생성합니다.

## 접근

기본 URL: `http://localhost:8080`

## 탐색

### 주요 섹션

1. **대시보드** - 시스템 개요 및 지표
2. **생명체** - 실리콘 생명체 관리
3. **채팅** - 생명체와 상호작용 (파일 업로드, 실시간 SSE 지원)
4. **채팅 기록** - 실리콘 생명체의 채팅 기록 보기 (세션 목록, 메시지 세부정보)
5. **작업** - 작업 관리 (개인 작업)
6. **타이머** - 타이머 설정 (만들기, 일시중지, 실행 기록)
7. **설정** - 시스템 설정 (AI 클라이언트, 지역화)
8. **권한** - 접근 제어 (ACL 관리, 권한 쿼리)
9. **로그** - 시스템 로그 (레벨별 필터링, 시간 범위 쿼리)
10. **감사** - Token 사용 및 감사 추적
11. **메모리** - 생명체 메모리 (타임라인 뷰, 고급 필터링)
12. **지식** - 지식베이스 (트리플 관리, 경로 발견)
13. **코드 브라우저** - 코드 탐색 (파일 트리, 구문 강조)
14. **코드 에디터** - 호버 힌트가 포함된 코드 편집 (Monaco Editor)
15. **프로젝트** - 프로젝트 관리 (워크스페이스, 작업, 작업 노트)
16. **실행기** - 실행기 관리 (디스크, 네트워크, 명령줄)
17. **도움말** - 도움말 문서 시스템 (다국어 지원, 주제 검색)
18. **정보** - 시스템 정보 및 버전

```javascript

};
```

        return ":root { --primary: #color; }";
    }
}

- 세션 목록 표시
- 메시지 세부정보 보기

   - 세션 제목

- **컨트롤러**: `ChatHistoryController`

  - `ChatHistoryDetailView` - 메시지 세부정보
- **API 라우트**:
  - `/api/chat-history/{beingId}/conversations` - 세션 목록 가져오기

## 도움말 문서 시스템 (신규)

### 기능 개요

도움말 문서 시스템은 실리콘 생명체와 사용자를 위한 다국어 도움말 문서 지원을 제공합니다.

### 도움말 문서 사용

1. **도움말** 페이지로 이동
2. 도움말 주제 목록 보기:
   - 빠른 시작 가이드
   - 도구 사용 참고
   - 권한 관리 가이드
   - 문제 해결 매뉴얼
   - 개발 가이드
3. 주제 클릭하여 상세 내용 보기:
   - 구조화된 문서 내용 (Markdown 렌더링)
   - 다국어 지원 (시스템 지역화 설정 따름)
   - 관련 주제 추천
4. 검색 기능 사용하여 빠르게 찾기:
   - 키워드 검색 (중국어, 영어 지원)
   - 검색 결과 관련성 순 정렬

### 실리콘 생명체의 도움말 접근

실리콘 생명체는 `help` 도구를 통해 도움말 문서에 접근할 수 있습니다:
```json
{
  "action": "get_topics"
}
```

### 기술 구현

- **컨트롤러**: `HelpController`
- **도구**: `HelpTool`
- **API 라우트**:
  - `/api/help` - 도움말 주제 목록 가져오기
  - `/api/help/{topicId}` - 주제 세부정보 가져오기
  - `/api/help/search?q=keyword` - 도움말 문서 검색

## 프로젝트 워크스페이스 (신규)

### 기능 개요

프로젝트 워크스페이스는 프로젝트 관리, 작업 추적 및 작업 노트를 지원하는 구조화된 작업 환경을 제공합니다.

### 프로젝트 관리

1. **프로젝트 만들기**:
   - 프로젝트 이름 및 설명
   - 프로젝트 태그 (분류)
   - 프로젝트 상태 (진행 중, 완료됨, 보관됨)
2. **프로젝트 세부정보 보기**:
   - 프로젝트 기본 정보
   - 연관 작업 목록
   - 작업 노트 목록
   - 프로젝트 진행률 통계
3. **프로젝트 보관**: 히스토리 데이터 보존하지만 더 이상 활성 아님

### 작업 노트 (비공개)

실리콘 생명체의 개인 작업 노트, 일기장 유사:

1. **노트 만들기**:
   - 요약 (짧은 설명)
   - 내용 (Markdown 형식 지원)
   - 키워드 (검색용)
   - 자동 타임스탬프 기록
2. **노트 관리**:
   - 타임라인 순 탐색 (페이지 디자인)
   - 노트 검색 (키워드, 요약, 내용별)
   - 목차 생성 (노트 구조 빠른 탐색)
   - 노트 업데이트 및 삭제
3. **권한 제어**:
   - 기본 비공개, 생명체 자신만 접근
   - 실리콘 큐레이터는 모든 노트 관리 가능

### 기술 구현

- **컨트롤러**: `WorkNoteController`
- **도구**: `WorkNoteTool`, `ProjectTool`, `ProjectWorkNoteTool`
- **API 라우트**:
  - `/api/worknotes` - 작업 노트 목록 가져오기
  - `/api/worknotes/{id}` - 노트 세부정보 가져오기
  - `/api/worknotes/search?q=keyword` - 노트 검색
  - `/api/worknotes/directory` - 노트 목차 생성
  - `/api/projects` - 프로젝트 관리 API

## 문제 해결

### 연결할 수 없음

**확인**:
- 서버 실행 중
- 포트 8080 차단되지 않음
- 방화벽 설정

### SSE 작동 안 함

**확인**:
- 브라우저가 SSE 지원
- 프록시가 SSE 버퍼링하지 않음
- 네트워크 안정성
