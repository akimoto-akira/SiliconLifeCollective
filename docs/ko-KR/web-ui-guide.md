# Web UI 가이드

## 개요

Web UI는 silicon being 관리, 시스템 상태 모니터링, AI 에이전트 상호작용을 위한 포괄적인 인터페이스를 제공합니다.

## 접근

기본 URL: `http://localhost:8080`

## 탐색

### 주요 섹션

1. **대시보드** - 시스템 개요 및 메트릭
2. **Beings** - silicon being 관리
3. **채팅** - being과 상호작용
4. **작업** - 작업 관리
5. **타이머** - 타이머 설정
6. **설정** - 시스템 설정
7. **권한** - 접근 제어
8. **로그** - 시스템 로그
9. **감사** - 토큰 사용량 및 감사 추적
10. **메모리** - being 메모리
11. **지식** - 지식 기반
12. **코드 브라우저** - 코드 탐색
13. **코드 에디터** - 호버 힌트가 있는 코드 편집
14. **프로젝트** - 프로젝트 관리
15. **Executor** - Executor 관리

## 대시보드

### 기능

- 시스템 성능 메트릭 (CPU, 메모리, 가동 시간)
- Being 상태 개요
- AI 사용 통계
- 빠른 작업

### 실시간 업데이트

실시간 데이터를 위한 SSE (Server-Sent Events) 사용:

```javascript
const dashboard = new EventSource('/api/dashboard/events');
dashboard.onmessage = (event) => {
    const data = JSON.parse(event.data);
    updateMetrics(data);
};
```

## Being 관리

### Being 목록

모든 being 표시:
- 이름 및 ID
- 현재 상태 (Running/Stopped/Error)
- 소울 파일 링크
- 빠른 작업 (시작/중지/설정)

### Being 상세

- 전체 설정
- 소울 파일 에디터
- 작업 기록
- 메모리 뷰어
- 성능 메트릭

### Being 생성

1. **새 Being 생성** 클릭
2. 입력:
   - 이름
   - 소울 내용 (Markdown 에디터)
   - 초기 설정
3. **생성** 클릭

## 스킨 시스템

### 사용 가능한 스킨

1. **Admin** - 전문 관리 인터페이스
2. **Chat** - 대화 중심 설계
3. **Creative** - 창의적이고 예술적인 스타일
4. **Dev** - 개발자 지향 레이아웃

### 스킨 전환

1. **설정** 클릭 (톱니바퀴 아이콘)
2. **스킨** 선택
3. 원하는 스킨 선택
4. 인터페이스 즉시 업데이트

### 사용자 정의 스킨

`ISkin` 구현하여 사용자 정의 스킨 생성:

```csharp
public class MySkin : ISkin
{
    public string Name => "MySkin";
    
    public string GetCss()
    {
        return ":root { --primary: #color; }";
    }
}
```

### 기능
- 실리콘 생명체 채팅 히스토리 탐색

- 타임라인 보기

### 기술 구현
- **컨트롤러**: `ChatHistoryController`
- **ViewModel**: `ChatHistoryViewModel`
- **뷰**:
  - `ChatHistoryListView` - 대화 목록
  - `ChatHistoryDetailView` - 메시지 상세
- **API 라우트**:
  - `/api/chat-history/{beingId}/conversations`
  - `/api/chat-history/{beingId}/conversation/{conversationId}`

## 파일 업로드

### 기능 설명
- 파일 소스 대화상자를 통한 파일 선택 지원
- 채팅 인터페이스에 파일 업로드 통합
- 파일 메타데이터 관리 (이름, 크기, 타입)
- 업로드된 파일에 대한 액세스 권한 제어 구현
- 업로드된 파일의 보안 검증 및 처리 로직 포함
- 여러 로컬라이제이션 파일에서 업로드 관련 현지화 문자열 추가:
  - 영어, 중국어, 광둥어, 일본어, 한국어 현지화 파일에 업로드 버튼 및 파일 소스 대화상자 텍스트 추가

### 파일 소스 대화상자 구현 세부사항:
- 파일 선택 버튼 클릭 시 대화상자 표시 로직 구현 (약 80줄)
- 대화상자 상태 관리 (열기/닫기/파일 선택)
- 선택된 파일 정보 표시 및 확인/취소 기능

## 로딩 표시기

### 채팅 페이지 로딩 표시기 구현:
- 채팅 페이지에 로딩 상태 표시기 추가 (50줄 추가)
- AI 응답 생성 중 시각적 피드백 제공

### 큐레이터 세션 자동 선택:
- 채팅 페이지 로드 시 큐레이터 세션 자동 선택 기능 구현 (10줄 추가)
- 사용자 경험 향상을 위한 자동 포커스 설정

### 연결 불가

**확인**:
- 서버 실행 중
- 포트 8080 차단되지 않음
- 방화벽 설정

### SSE 작동 안 함

**확인**:
- 브라우저 SSE 지원
- SSE 프록시 버퍼링 없음
- 네트워크 안정성

### 느린 성능

**최적화**:
- 로그 상세도 축소
- 오래된 감사 데이터 정리
- 시스템 리소스 확인
