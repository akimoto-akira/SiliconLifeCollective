## Seznamte se s Vašimi Digitálními Společníky

### Co jsou Křemíkové Bytosti?

Křemíkové bytosti jsou autonomní AI agenti, kteří:
- 🧠 **Přemýšlejí** - Mají vlastní osobnost a rozhodovací schopnosti
- 💬 **Komunikují** - Konverzují přirozeně a kontextově
- 🛠️ **Jednají** - Používají nástroje k provádění skutečných úkolů
- 📚 **Učí se** - Vyvíjejí se a zlepšují v čase
- 🔄 **Evoluuji** - Mohou přepisovat svůj vlastní kód

### Proč Silicon Life Collective?

#### Pro Uživatele
- **24/7 Asistence** - Vaši digitální společníci nikdy nespí
- **Personalizace** - Každá bytost má jedinečnou osobnost
- **Multi-Tasking** - Více bytostí pracuje současně na různých úkolech
- **Dlouhodobá Paměť** - Pamatuji si vaše preference a historii

#### Pro Vývojáře
- **Open Source** - Plně transparentní a rozšiřitelný
- **Plugin System** - Snadné přidávání nových funkcí
- **Multi-AI Support** - Podpora Ollama, DashScope a dalších
- **Self-Evolving** - AI mohou vylepšovat svůj vlastní kód

### Začněte Dnes

#### 1. Nainstalujte
```bash
git clone https://github.com/akimoto-akira/SiliconLifeCollective.git
cd SiliconLifeCollective
dotnet build
```

#### 2. Nakonfigurujte
Nastavte váš preferovaný AI backend (Ollama nebo Cloud).

#### 3. Spusťte
```bash
dotnet run --project src/SiliconLife.Default
```

#### 4. Vytvořte Vaši První Bytost
Definujte osobnost v soul.md a sledujte, jak ožívá!
