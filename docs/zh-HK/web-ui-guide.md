# Web UI 指南

[English](../en/web-ui-guide.md) | [中文](../zh-CN/web-ui-guide.md) | **繁體中文** | [Español](../es-ES/web-ui-guide.md) | [日本語](../ja-JP/web-ui-guide.md) | [한국어](../ko-KR/web-ui-guide.md) | [Čeština](../cs-CZ/web-ui-guide.md)

## 概述

Web UI 提供了一個全面的界面，用於管理硅基生命体、監控系統狀態和與 AI 智能体交互。

## 訪問

默認 URL：`http://localhost:8080`

## 導航

### 主要部分

1. **儀表板** - 系統概览和指标
2. **生命体** - 管理硅基生命体
3. **聊天** - 與生命体交互
4. **工作** - 工作管理
5. **定时器** - 定时器設定
6. **設定** - 系統設定
7. **權限** - 訪問控制
8. **記錄** - 系統記錄
9. **稽核** - Token 使用和稽核跟踪
10. **記憶** - 生命体記憶
11. **知识** - 知识程式庫
12. **程式碼瀏覽器** - 程式碼探索
13. **程式碼编辑器** - 带悬浮提示的程式碼编辑
14. **項目** - 項目管理
15. **執行器** - 執行器管理

## 儀表板

### 功能

- 系統效能指标（CPU、記憶體、執行時間）
- 生命体狀態概览
- AI 使用統計
- 快速操作

### 实时更新

使用 SSE（伺服器發送事件）获取实时資料：

```javascript
const dashboard = new EventSource('/api/dashboard/events');
dashboard.onmessage = (event) => {
    const data = JSON.parse(event.data);
    updateMetrics(data);
};
```

## 生命体管理

### 生命体列表

顯示所有生命体，包含：
- 名称和 ID
- 當前狀態（執行中/已停止/錯誤）
- 靈魂檔案链接
- 快速操作（啟動/停止/設定）

### 生命体详情

- 完整設定
- 靈魂檔案编辑器
- 工作歷史
- 記憶查看器
- 效能指标

### 建立生命体

1. 点擊**建立新生命体**
2. 填寫：
   - 名称
   - 靈魂內容（Markdown 编辑器）
   - 初始設定
3. 点擊**建立**

## 皮肤系統

### 可用皮肤

1. **Admin** - 專業管理界面
2. **Chat** - 以對话為中心的設計
3. **Creative** - 創意和藝術風格
4. **Dev** - 面向開發者的布局

### 切换皮肤

1. 点擊**設定**（齒轮图标）
2. 选择**皮肤**
3. 选择所需皮肤
4. 界面立即更新

### 自定義皮肤

通過實現 `ISkin` 建立自定義皮肤：

```csharp
public class MySkin : ISkin
{
    public string Name => "MySkin";
    
    public string GetCss()
    {
        return ":root { --primary: #color; }";
    }
}
```

## 權限管理

### 查看權限

- 列出所有權限規則
- 按使用者或資源過濾
- 查看過期日期

### 添加權限規則

1. 点擊**添加規則**
2. 設定：
   - 使用者
   - 資源（例如 `disk:read`）
   - 允許/拒絕
   - 持续時間
3. 儲存

### 稽核跟踪

查看所有權限決策：
- 時間戳
- 使用者
- 資源
- 決策
- 原因

- 訊息详情查看

## 檔案上傳

### 功能说明
- 支援通過檔案來源對話框選擇檔案
- 將檔案上傳集成到聊天介面
- 檔案元資料管理（名稱、大小、類型）
- 實現上傳檔案的訪問權限控制
- 包含上傳檔案的安全驗證和處理邏輯
- 在多個本地化檔案中添加上傳相關本地化字符串：
  - 英文、中文、粵語、日語、韓語本地化檔案中添加上傳按鈕和檔案來源對話框文本

### 檔案來源對話框實現细节：
- 實現點擊檔案選擇按鈕時顯示對話框的邏輯（約 80 行）
- 對話框狀態管理（開啟/關閉/檔案選擇）
- 顯示所選檔案資訊及確認/取消功能

## 載入指示器

### 聊天頁面載入指示器實現：
- 爲聊天頁面添加載入狀態指示器（50 行新增）
- 在 AI 回應生成過程中提供視覺回饋

### 主理人會话自動選擇：
- 實現聊天頁面載入時自動選擇主理人會话的功能（10 行新增）
- 設置自動焦點以提升使用者體驗

### 无法連接

**檢查**：
- 伺服器正在執行
- 端口 8080 未被阻止
- 防火墙設定

### SSE 不工作

**檢查**：
- 瀏覽器支援 SSE
- 无代理缓衝 SSE
- 網路穩定性

### 效能慢

**最佳化**：
- 降低記錄详细程度
- 清理舊稽核資料
- 檢查系統資源
