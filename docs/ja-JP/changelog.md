## この変更履歴について

### プロジェクトの起源

- 本プロジェクトは2026年3月20日に始まりました。
- このプロジェクト以前には、アーキテクチャ設計が不合理だったため、複数のAIプラットフォームと統合できなかった検証デモが失敗しました。

### 使用されたAI IDEツール

#### Kiro（Amazon AWS）
- プロジェクトは最初にKiroによってメンテナンスされ、Specモードを使用して起動されました。
- KiroはAmazon AWSによって構築されたエージェンティックAI開発環境です。
- Code OSS（VS Code）に基づいており、VS Code設定とOpen VSX互換プラグインをサポートします。
- 構造化AIコーディングのための仕様駆動開発ワークフローを備えています。

#### Comate AI IDE / 文心快码（Baidu）
- コピーライティングとドキュメント作業のために時々使用されます。
- Comate AI IDEは、百度文心が2025年6月23日にリリースしたAIネイティブ開発環境ツールです。
- 業界初のマルチモーダル、マルチエージェント協業AI IDEです。
- デザインからコードへの変換とフルプロセスAI支援コーディング機能が含まれます。
- 百度の文心4.0 X1 Turboモデルによって駆動されます。

#### Trae（ByteDance）
- このプロジェクトは大部分の時間、主にTraeを使用してメンテナンスされました。
- TraeはByteDanceのシンガポール子会社SPRING PTEによって開発されたAI IDEです。
- 独立してソフトウェアソリューションを構築できる10x AIエンジニアとして機能します。
- インテリジェント生産性ツール、柔軟な開発リズム適応、協業プロジェクト配信機能を備えています。
- 設定可能なエージェントシステムを備えたエンタープライズグレードのパフォーマンスを提供します。

#### Qoder（Alibaba）
- 2026年4月18日以降、本プロジェクトはQoderを使用してメンテナンスされています。
- Qoderはソースコード分析とドメインドキュメント生成に優れており、複雑なコードベースの理解において卓越した能力を示しています。
- ゼロ計算コストの価格モデルで動作し、自動化されたドキュメント処理と日常のタスク処理において非常にコスト効率が高くなっています。
- 実際のソフトウェア開発のために設計されたAI駆動のエージェンティックコーディングプラットフォームです。
- インテリジェントコード生成、会話ベースのプログラミング、高度なコンテキスト分析エンジン、マルチエージェントコラボレーション機能を備えています。
- 最小限のリソース消費で深いコード理解を提供し、長期的なプロジェクトメンテナンスと知識蓄積に理想的です。

### 要件ドキュメント

- 本プロジェクトの要件ドキュメントは公開されていません。
- 要件は12以上の国際的なAIプラットフォームと大規模モデルシリーズによって繰り返し検証され、人間にはほとんど理解できない2,000行以上のユーザーストーリー駆動の要件ドキュメントが作成されました。
