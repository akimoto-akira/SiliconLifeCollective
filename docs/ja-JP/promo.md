# SiliconLifeCollective プロモーション

## プロジェクト概要

**SiliconLifeCollective** は、AI エージェントのためのオープンプラットフォームです。自律的なシリコンビーイングを作成、管理、対話することができます。

---

## キー機能

### 🤖 AI エージェント
- 自律的なシリコンビーイング
- 自然言語対話
- ツール使用能力
- 長期記憶

### 🛠️ 拡張可能
- プラグイン式ツールシステム
- カスタム AI クライアント
- スキンテーマ
- カレンダーシステム（32種類）

### 🔒 セキュア
- 5レベル権限システム
- 監査ログ

- 動的コンパイルセキュリティ

### 🌐 マルチ言語
- 5言語サポート
- ローカライズされた UI

### 💻 開発者フレンドリー
- オープンソース（Apache 2.0）
- .NET 9
- 明確なドキュメント
- 活発なコミュニティ

## 技術ハイライト

### ボディ-ブレインアーキテクチャ

独自の Body-Brain 分離設計：
- **ボディ**：状態とトリガーを管理
- **ブレイン**：AI インタラクションを処理

### プラグイン式ツール

リフレクションベースの自動検出：
- ツールを追加するだけで自動登録
- カスタムツールを簡単に作成
- 管理者専用ツール

### 包括的カレンダー

32 種類のカレンダーシステム：
- グレゴリオ暦
- 中国旧暦
- イスラム暦
- 歴史的カレンダー
- 民族文化カレンダー

## クイックスタート

```bash
# リポジトリをクローン
git clone https://github.com/akimoto-akira/SiliconLifeCollective.git

# ビルド
cd SiliconLifeCollective
dotnet build

# 実行
cd src/SiliconLife.Default
dotnet run

# Web UI にアクセス
# http://localhost:8080
```

## FAQ

### Q: 無料で使えますか？
A: はい！完全にオープンソースで無料です。

### Q: どの AI モデルをサポートしていますか？
A: 現在 Ollama（ローカル）と DashScope（クラウド）をサポートしています。カスタムクライアントも追加可能です。

### Q: 技術的な専門知識が必要ですか？
A: 基本的な使用には必要ありません。カスタマイズには .NET の知識が役立ちます。

### Q: データはどこに保存されますか？
A: ローカルファイルシステムに保存されます。クラウド同期は計画中です。

### Q: 貢献するには？
A: [貢献ガイド](contributing.md)を参照してください。あらゆる形の貢献を歓迎します！
